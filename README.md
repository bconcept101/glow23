# Glow 23

Glow 23 is a Telegram entertainment number game with a live web page reveal.

The live page allows members to watch the game status, follow the daily round schedule, view the animated chamber, see revealed results, and search previous results.

## Project Type

Telegram entertainment number game with a live web page reveal.

## Main Game Terms

- Game name: Glow 23
- Main number: Glow Number
- Bonus number: Glow Ball
- Glow Number range: 1–23
- Glow Ball range: 1–20

## Locked Reserve

- 7% cash-out reserve
- 3% hosting and maintenance reserve
- 10% total reserve deducted from winning returns

## Locked Member Rule

Members may purchase multiple Glow Numbers per round.

Each individual Glow Number can be purchased up to three times per member, per round.

Example:

- Glow Number 7 — up to 3 times
- Glow Number 12 — up to 3 times
- Glow Number 19 — up to 3 times

## Daily Round Schedule

Glow 23 runs three automatic rounds per day.

| Round | Reveal / Play Window |
| --- | --- |
| Round 1 | 12:00 PM – 1:00 PM |
| Round 2 | 3:00 PM – 5:00 PM |
| Round 3 | 7:00 PM – 10:00 PM |
