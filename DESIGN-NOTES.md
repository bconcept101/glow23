# Glow 23 Design Notes

This file documents the visual and interface direction for the Glow 23 live game page.

## Main Design Direction

Glow 23 should feel like a sleek, colorful, 3D entertainment game interface.

The design should feel:

- Premium
- Colorful
- Smooth
- Playful
- Live
- Animated
- Easy to understand
- Mobile-friendly

## Visual Style

The interface should have a rich 3D look with strong colors.

The design inspiration should feel similar to a classic colorful arcade or platform game style, while still looking modern and clean.

The page should use:

- Rich glowing colors
- Gold highlights
- Purple and pink accents
- Bright blue effects
- Green live/open status indicators
- Red closed/cutoff status indicators
- Dark background for contrast
- Rounded panels
- Soft shadows
- Glass-style effects
- 3D button effects

## Main Chamber Concept

The center of the page should feature a chamber-style drawing machine.

The chamber should show:

- A round glass chamber
- Balls moving or fumbling inside
- A center Glow 23 machine core
- A shiny glass reflection
- A colorful stage-like appearance

The chamber should look exciting enough for members to watch while waiting for results.

## Drawn Ball Reveal

When a ball is drawn, one ball should pop up at the top of the chamber.

The drawn ball should:

- Be large and easy to see
- Use a strong gold/orange 3D look
- Show the selected Glow Number clearly
- Animate with a pop effect
- Feel like a live reveal moment

## Live Page Sections

The live interface should include:

1. Header
2. Side menu
3. Current round status
4. Countdown timer
5. Star placement cutoff status
6. 3D chamber
7. Drawn ball display
8. Today’s results
9. Previous results search
10. Rules section
11. Footer

## Side Menu

The side menu should give members quick access to important sections.

The side menu should include links to:

- Live Game
- Today’s Results
- Previous Results
- Game Rules

The side menu should also include a search box where members can search previous results.

Search should support:

- Date
- Round
- Glow Number
- Glow Ball
- Timestamp

## Today’s Results

Today’s results should remain visible on the page even after the next round starts.

Each result should show:

- Round number
- Glow Number
- Glow Ball
- Timestamp
- Pending status before reveal

## Previous Results Search

The previous results search should allow members to look back at older Glow 23 results.

Each previous result should show:

- Date
- Round
- Glow Number
- Glow Ball
- Timestamp
